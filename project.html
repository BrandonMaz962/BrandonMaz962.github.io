Projects

I have worked on many projects of varying sizes and in many languages. Below are some of my favorite projects that I have worked on that are also the most complete and functioning. Click on the project name to learn more.

Python

  • Frog Jump

Frogger style game where the player controls a frog sprite and using the arrow keys to move forward, left, right, and down tries to cross the road and reach the water on the other side. Collecting flies will lower your final time but getting hit by a vehicle will cause a game over. I made this game using python3 and pygame libraries.

Javascript / HTML

  • Block Breaker

Breakout style game where the player controls the platform on the bottom of the play area using the arrow keys to move left or right. Using the platform to bounce the ball the player tries to destroy all of the blocks. If all the blocks are destroyed you win however, if the ball falls off the screen then its game over. I made this game using a combination of javascript, html, and phaser3 libraries.
